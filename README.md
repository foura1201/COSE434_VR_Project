# COSE434_VR_Project
> 2022 가을학기 COSE434 게임프로그래밍 VR Game Project
## 프로젝트 이름
```
Escape Magic Room
```
## 팀원
* 고려대 컴퓨터학과 [정세진](https://github.com/asjay18)
* 고려대 컴퓨터학과 [허은진](https://github.com/foura1201)
## 개발 환경
* Unity 2021.3.9f1
* VR: [Meta] Oculus Quest 2
